# Anti-Matter - IPA Sideloading Store

A mobile app store for sideloading IPA files on iOS devices.

## Features

### Home Screen
- Anti-Matter branding with atom logo
- "How to Sideload" tutorial card
- Latest Apps horizontal carousel
- Top Apps grid with ranking badges
- Donate/Support section
- iOS version compatibility info

### Search Screen
- Search bar for apps and developers
- Category grid with icons:
  - Games, Entertainment, Health, Weather
  - Finance, Home, Music, Sports
  - Education, Travel, Utilities, Social

### Category Screen
- Filtered app listings by category
- Ranked list with app details

### App Detail Screen
- App icon, name, developer
- Rating and download count
- Download IPA button
- Install Certificate option (when available)
- App info (size, version, iOS requirement)
- Full description
- Social links (Twitter, Website, App Store)

### Developer Portal
- Upload app icon
- App name, description, version
- Category selection
- IPA file upload
- Social links (optional)
- Submission guidelines

### Settings
- Notifications toggle
- Appearance (follows system)
- iOS Compatibility info
- Support (Patreon, Discord)
- Rate App
- Downloaded Apps management
- Clear Cache
- Privacy Policy & Terms

## Tech Stack

- React Native + Expo SDK 53
- Expo Router for navigation
- Zustand for state management
- NativeWind (TailwindCSS)
- Lucide icons
- expo-image for optimized images
- expo-haptics for feedback
- expo-linear-gradient for gradients

## Color Scheme

- Primary: Purple (#8B5CF6, #7C3AED)
- Dark background: #0A0A0F
- Dark cards: #16161F
- Accent: Pink gradient for donate section
